# dripdrop

![pepe chillin](pepe_on_rain.png)

Listen to endless rain from your terminal

## Requirements

[go](https://go.dev)

## Installation

Build the package using go compiler: `go build`  
I have a plan to make this into an AUR package

## Usage

Just run the binary  
Press space to pause and resume  
Press q to quit
